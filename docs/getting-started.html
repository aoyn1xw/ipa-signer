Getting Started

Run locally with Node.js or use Docker for a repeatable environment.

Local setup

npm install
node app.js

Server defaults to http://localhost:3000.

Production checklist

  • Set PUBLIC_DOMAIN to your public HTTPS URL.
  • Use a reverse proxy (Nginx/Caddy/Cloudflare Tunnel) for TLS.
  • Tune rate limit values for your traffic profile.
  • Monitor logs/error.log and logs/combined.log.

Environment variables

NameDefaultPurpose
PORT3000HTTP server port
PUBLIC_DOMAINhttps://yourdomain.comBase URL for install and plist links
CYAN_CMDcyanPath or command for cyan executable
RATE_LIMIT_WINDOW_MS900000Rate-limit time window
RATE_LIMIT_MAX100Max requests per window
LOG_LEVELinfoWinston log level

Docker setup

docker build -t ipa-signer .
docker run --rm -p 3000:3000 \
  -e PUBLIC_DOMAIN=https://your-domain.example/ \
  ipa-signer

Quick health checks

curl -I http://localhost:3000/
curl -I http://localhost:3000/style.css
