Basic Customization

This page covers the most common changes without modifying business logic.

1) Change server port

Set PORT at runtime.

PORT=8080 node app.js

2) Change public install domain

Set PUBLIC_DOMAIN. Always use HTTPS and include your final host.

PUBLIC_DOMAIN=https://sign.example.com/ node app.js

3) Tune rate limiting

RATE_LIMIT_WINDOW_MS=900000
RATE_LIMIT_MAX=100

Example for stricter settings:

RATE_LIMIT_WINDOW_MS=60000
RATE_LIMIT_MAX=20

4) Change cyan executable path

If cyan is not on PATH:

CYAN_CMD=/usr/local/bin/cyan node app.js

5) Update UI text/branding

Edit the upload UI in index.html and style it in style.css.
