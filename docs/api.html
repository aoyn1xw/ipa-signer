API Reference

Built-in endpoints currently implemented in app.js.

POST /sign

Signs an IPA and returns install URLs.

Content-Type

multipart/form-data

Required files

FieldTypeNotes
ipafile.ipa
p12file.p12
mobileprovisionfile.mobileprovision

Optional fields

FieldMeaning
p12_passwordPassword for the P12
adv_nameSet app display name (cyan -n)
adv_versionSet app version (cyan -v)
adv_bundle_idSet bundle ID (cyan -b)
adv_min_osSet min iOS version (cyan -m)
adv_remove_supported_devicesRemove UISupportedDevices
adv_no_watchRemove watch app
adv_fakesignFakesign binaries
adv_thinThin binaries to arm64
adv_remove_extensionsRemove all extensions
adv_remove_encryptedRemove encrypted extensions

Success response

{
  "installLink": "https://domain/install/12345",
  "directInstallLink": "itms-services://?action=download-manifest&url=https://domain/plist/app.plist"
}

Error response

{
  "error": "Signing failed",
  "details": "..."
}

GET /install/:id

Returns a simple install page for a signed build. Link expires after one hour.

Static routes

  • GET / - Upload UI
  • GET /style.css - UI stylesheet
  • GET /signed/* - Signed IPA static files
  • GET /plist/* - Manifest plist static files
