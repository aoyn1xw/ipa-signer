Overview

IPA Signer is a web service that signs iOS IPA files using your provisioning profile and certificate. It also supports advanced modifications through cyan before signing.

What this repo includes

  • Express server and upload flow in app.js
  • Worker-thread signing integration in zsign-worker.js
  • Web UI for uploads in index.html and style.css
  • Dockerized runtime with zsign and cyan

Core flow

  1. Client uploads .ipa, .p12, and .mobileprovision to POST /sign.
  2. Optional advanced options are applied through cyan.
  3. IPA is signed via zsign in a worker thread.
  4. Server returns install URLs (page URL + direct itms-services link).
